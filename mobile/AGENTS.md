# Expo HAS CHANGED

Read the exact versioned docs at https://docs.expo.dev/versions/v54.0.0/ before writing any code.

This app is pinned to SDK 54 on purpose: the App Store / Play Store builds of
Expo Go only support SDK 54 (Apple has newer Expo Go versions stuck in review
since May 2026), and the team tests via store-installed Expo Go. Do not
upgrade the SDK until Expo Go in the stores supports the target version or
the team switches to development builds.
